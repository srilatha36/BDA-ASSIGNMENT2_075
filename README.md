# 📊 Big Data Analytics Assignment (PySpark)

## 📌 Overview

This project demonstrates the implementation of **machine learning techniques using PySpark MLlib**.
The assignment covers three major tasks:

* Classification
* Clustering
* Recommendation System

Each task is implemented using real-world datasets with proper preprocessing, evaluation metrics, and visualizations.

---

## 🚀 Technologies Used

* Python
* PySpark (MLlib)
* Pandas
* Matplotlib
* Google Colab

---

## 📂 Project File

* `BDA.ipynb` → Contains all implementations (Classification, Clustering, Recommendation)

---

# 🔍 1. Classification (Titanic Dataset)

## 🎯 Objective

Predict whether a passenger survived or not.

## ⚙️ Algorithm Used

* Logistic Regression

## 📊 Features Used

* Pclass
* Sex
* Age
* Fare

## 📈 Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

## 📊 Visualization

* Confusion Matrix Heatmap
* Age vs Fare Scatter Plot

---

# 🔍 2. Clustering (Iris Dataset)

## 🎯 Objective

Group similar data points into clusters.

## ⚙️ Algorithm Used

* K-Means Clustering

## 📈 Evaluation Metric

* Silhouette Score

## 📊 Visualization

* Cluster Scatter Plot (Sepal Length vs Petal Length)

---

# 🔍 3. Recommendation System (MovieLens Dataset)

## 🎯 Objective

Recommend movies to users based on their ratings.

## ⚙️ Algorithm Used

* ALS (Alternating Least Squares)

## 📈 Evaluation Metric

* RMSE (Root Mean Square Error)

## 📊 Visualization

* Ratings Distribution Histogram

---

## ⚙️ Setup Instructions

1. Open Google Colab
2. Install PySpark:

   ```bash
   pip install pyspark
   ```
3. Run all cells sequentially

---

## 📈 Results

* The classification model achieved good accuracy with balanced precision and recall
* The clustering model formed meaningful clusters with a good silhouette score
* The recommendation system generated personalized movie suggestions with low RMSE

---

## 🎯 Conclusion

This project shows how **PySpark MLlib** can be used to perform scalable machine learning tasks efficiently.
The use of multiple evaluation metrics and visualizations improves model interpretability and performance analysis.
